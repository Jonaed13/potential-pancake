## 2025-02-18 - Discoverability in TUIs
**Learning:** Terminal interfaces lack the visual cues of GUIs (like buttons or menus), making keyboard shortcuts hard to discover. Users often rely on memory or external documentation.
**Action:** Always include a built-in "Cheat Sheet" or Help modal accessible via standard keys like `?` or `F1` to improve learnability without cluttering the main interface.
